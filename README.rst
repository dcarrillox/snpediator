==========
snpediator
==========

Queries SNPedia to retrieve basic SNP information:

- Gene

- Chromosome

- Genotypes

- Stabilized Orientation

- Position

- Reference Version



Install
=======


   pip install snpediator


Run
===


   snpediator -h
   
   snpediator -r <RSID>
